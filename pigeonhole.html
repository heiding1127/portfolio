Overview

Pigeonhole is a cloud storage app for educational purposes. A platform to store, collaborate, share ideas, and communicate anytime, anywhere. This app is designed specifically to define a new way to learn with the most fun, effective, and easy to use educational assessment tools for students and educators.

Problems

  • 1. The organization can be complicated with so many files for different class
  • 2. Sharing file process can be confusing
  • 3. Students find it hard to communicate with their educators outside of class.
  • 4. There are too many deadlines.

Solutions

  • 1. Files can be saved into each class folder to make sure student can organize
  • 2. Create a simple file sharing process
  • 3. Instead of sending an email, students and educators can communicate through messaging with each other.
  • 4. Introduce assignment status tag and event & activity reminder

Competitive Analysis

To see where Pigeonhole will sit in the market, I needed to dig deeper into the world of cloud storage and organization apps. I did a competitive analysis for Google classroom, Pinterest, and Dropbox.

Overall, I think they all are good in different ways and serving different purposes. Google classroom is focusing on education purposes for class but the service might be too general that might be hard to a bigger scale class to use it or any professor who has multiple classes. The advantage is that with Google, everything is linked in together and saved automatically. Pinterest serves a purpose for researching or gathering ideas platforms like cooking, decorating, fashion, gardening, and other topics of interest to women. Most of the users do not use it as a storage platform. Dropbox is one of the most popular cloud storage services. It’s easy to use and able to access with any devices but they are pricey compared to other similar services such as google drive.

I noticed that there’s no actual cloud storage app that can be combining all of these functions. What if Pigeonhole can offer users all these functions at once? Will it create a new learning experience for students and help educators to be more successful?

User Persona

Katy G

39 years old | Real-Estate Agent | Los Angeles, CA | married

“AS A MOTHER OF TWO, I WANT TO PROVIDE THE BEST EDUCATION TO MY KIDS.”

Goal

Katy cares about her kids' education. She spends lots of time with her kids to help them on their homework. However, it is not easy for her because she may not understand all the questions, and the only way to get help from the teacher is to wait until they are in class. She thinks it would be great if there was a tool for her kids to ask questions whenever they needed.

Frustration

  • 1. Hard to find out due-date
  • 2. Hard to communicate with the teacher as parents
  • 3. Do not know the expectation of the teacher

Christina E

21 years old | College Student | Orange county, CA | single

“I WANT TO BE ABLE TO ORGANIZE MY FILE ANYTIME, ANYWHERE.”

Goal

Christina is at the end of her college undergraduate journey with lots of assignments, papers, and tests to go. She also has a few group projects, a large number of contents, or notes in her current cloud storage. It’s hard for her to look for the files effectively and always having problems with file accessibility with her professor. She is always busy with the other things so she needs to have access to file anytime on the go so that she can make sure everything works well.

Frustration

  • 1. Hard to find a file for different classes
  • 2. Time management is becoming hard because a lot of things going on
  • 3. File accessibility problem

User Stories

Based on the information gathered from the surveys, and competitive analysis I created user stories to highlight the main features of the product. The user stories were then converted into User flows done in illustrator, to illustrate how the user would interact with the product and accomplish those goals.


As a user, I want to be able to/ have:
  • 1. Upload files (videos, images, PDFs, etc.)
  • 2. Organize that content using things like categories, tags, groups, and/or folders
  • 3. Create content (notes, documents, or spreadsheets)
  • 4. Share an item or folder to an individual or a group
  • 5. Access in any type of devices
  • 6. A communication tool, chat room
  • 7. Event and activity reminder

Branding

Pigeonhole is a fun, professional and trustworthy brand. I believe colorful and fun illustrations will help students to be more engaged and interested in using this program. It builds a personality and comprehensive look + feel - it evokes a sense of education like "building blocks".

Pigeonhole is a metaphor of this business, students organizing things in their own locker or shelf. It’s a unique name and lends itself to a really great visual. This bold line pigeon head as the icon with the round logotype is the simplest way to represent the brand - Pigeon and also metaphor of kids. The round logotype is easy to read and memorable as well.
